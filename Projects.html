A Simple HTML and JS Game

Gamedev Canvas Workshop
Here's a simple game I made in HTML and Javascript!
You use the arrow keys to keep the ball off the
bottom of the screen! It uses simple drawing and
canvas updates, along with event handling for the
arrow keys being pressed. It also uses a textbox for
when you fail. My high score is 23, if you'd like to
try and beat it. I think the game gets boring after
a while so i may come back and update it with the
speed of the ball increasing!
